Para executar o programa, execute os passos a seguir:

• Vá ao endereço localhost/phpmyadmin/ e crie um novo usuário com o nome "fillipe", sem senha e com todos os privilégios.

• Crie um novo banco de dados com o nome "av2fillipe".

• Entre no banco de dados criado e clique em Importar. Selecione o arquivo "av2fillipe.sql" e clique em executar.

• Abra a página "home.html" através do localhost.
